Patrulaterul este un poligon cu patru laturi. Patrulaterele pot fi convexe sau concave. Patrulaterul convex este patrulaterul care are toate laturile în „exterior”, iar patrulaterul concav este patrulaterul care are două laturi în „interior”.

Elementele unui patrulater:

Laturile: [AB], [BC], [CD], [DA]

Unghiurile: ABC, BCD, CDA,DAB

Vârfurile: A, B, C, D.

Proprietate Suma măsurilor unghiurilor unui patrulater este egală cu 360°.

Perimetrul (P) unui patrulater este suma lungimilor laturilor.

P = AB+BC+CD+DA

II. Proprietatile patrulaterelor convexe

are laturile opuse paralele:

AB || CD si BC || AD;

Proprietati:

1. Laturile opuse sunt congruente:

`AB ≡ CD` si `BC ≡ AD`

2. Unghiurile opuse sunt congruente:

`hat (DAB) ≡ hat (BCD)` si `hat (ABC) ≡ hat (CDA)`

3. Unghiurile alăturate sunt suplementare:

`m(hat (DAB)) + m(hat (ABC)) = m(hat (BCD)) + m(hat (CDA)) = 180°`

4. Diagonalele se înjumătățesc:

`AO ≡ OC` si `BO ≡ OD`

Un patrulater este paralelogram dacă și numai dacă îndeplinește una din aceste condiții:

1. laturile opuse sunt congruente două câte două

2. are două laturi opuse paralele și congruente

3. unghiurile opuse sunt congruente

4. oricare două unghiuri alăturate sunt suplementare

5. diagonalele sale se înjumătățesc

Perimetrul P este:

`P = 2*l +2*L`

Area A, considerand unghiul este hat (DAB):

`A = AB * AD * sin hat (DAB)`

este paralelogramul cu un unghi drept;

AB || CD si BC || AD;

Proprietati:

1. Laturile opuse sunt congruente:

`AB ≡ CD` si `BC ≡ AD`

2.Laturi opuse sunt paralele:

AB || CD si BC || AD

3. Unghiurile sunt congruente si au masura de 90°:

`m(hat (DAB)) = m(hat (ABC)) = m(hat (BCD)) = m(hat (CDA)) = 90°`

4. Diagonalele se înjumătățesc:

`AO ≡ OC` si `BO ≡ OD`

Un paralelogram este dreptunghi dacă și numai dacă diagonalele sale sunt congruente:

`AC ≡ BD`

Perimetrul P este:

`P = 2*l +2*L`

Area A este considerata produsul dintre lungimea si latimea:

`A = L * l`

este paralelogramul cu două laturi consecutive congruente;

AB || CD si BC || AD;

Proprietati:

1. Laturile opuse sunt congruente:

`AB ≡ CD` si `BC ≡ AD`

2.Laturi opuse sunt paralele:

AB || CD si BC || AD

3. Unghiurile opuse sunt congruente :

`m(hat (DAB)) = m(hat (BCD))` si `m(hat (ABC)) = m(hat (CDA)) = 90°`

4. Diagonalele se înjumătățesc:

`AO ≡ OC` si `BO ≡ OD`

este patrulaterul ce are proprietatile unui dreptunghi și romb simultan.

Proprietati:

1. Toate laturile sunt congruente:

`AB ≡ CD ≡ BC ≡ AD`

2.Laturi opuse sunt paralele:

AB || CD si BC || AD

3. Unghiurile sunt congruente si au masura de 90°:

`m(hat (DAB)) = m(hat (ABC)) = m(hat (BCD)) = m(hat (CDA)) = 90°`

4. Diagonalele se înjumătățesc, sunt bisectoarele unghiurilor si sunt perpenticulare:

`AO ≡ OC` si `BO ≡ OD` si `AC ⊥ BD`
